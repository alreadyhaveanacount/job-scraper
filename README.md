# 💼 Job Scraper with personalized filters (Remote.com)

A job vacancy web-scraper, built with **BeautifulSoup**, **Pandas** and **Streamlit**, that scrapes [Remote.com](https://remote.com/jobs/all) based on personalized filters.

---

## 🚀 Funcionalities

- 📄 Scrapes Remote.com data
- 🔍 Filtros de busca:
  - Contract type (Full-time, Part-time, Contract)
  - Workplace (Remote, Hybrid, On-site)
  - Minimal seniority
  - Currency and minimal salary
  - Key-words
  - Travel frequency
- 💰 Calculates monthly average
- 📥 Exports data in **CSV**
- 👁️ Data visualization

---

## Libraries used

- [Python 3.10+](https://www.python.org/)
- [BeautifulSoup4](https://www.crummy.com/software/BeautifulSoup/)
- [Pandas](https://pandas.pydata.org/)
- [Streamlit](https://streamlit.io/)
- [Requests](https://docs.python-requests.org/)
